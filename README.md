🇱🇰 Sri Lanka Travel Website

A travel planning web application built with Python and React that helps users explore tourist destinations in Sri Lanka, book trains and hotels, and plan trips with location details and budget estimates in one platform.

✨ Features

🚆 Train booking for traveling across Sri Lanka

🏨 Hotel booking system

📍 Tourist location details and travel information

💰 Budget planning for trips

🗺️ Easy trip planning and travel recommendations

🛠️ Technologies Used

Frontend: React

Backend: Python

Version Control: Git & GitHub

The goal of this project is to simplify travel planning in Sri Lanka by providing a single platform where travelers can find destinations, book transportation and hotels, and manage their travel budget.
